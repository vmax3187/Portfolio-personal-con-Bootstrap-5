# Portfolio Profesional - Francisco Ángel Pérez Ponce 🚀

¡Bienvenido/a a mi repositorio de marca personal! Este proyecto consiste en un **Portfolio Digital Profesional** diseñado y desarrollado utilizando **Bootstrap 5**. Está orientado a mostrar mi trayectoria, competencias técnicas y proyectos como Desarrollador de Aplicaciones Web, garantizando una experiencia de usuario (UX) óptima, fluida y adaptativa.

---

## 🛠️ Características Principales y Componentes de UI

El sitio ha sido estructurado siguiendo las mejores prácticas de desarrollo frontend y los requisitos técnicos de Bootstrap 5:

*   **Dominio del Grid System:** Diseño 100% responsive optimizado para una visualización impecable en dispositivos móviles, tablets y pantallas de escritorio.
*   **Navegación Intuitiva (Navbar):** Menú lateral fijo interactivo con asistencia de *ScrollSpy* para detectar automáticamente la sección activa durante el desplazamiento.
*   **Proyectos en Tarjetas (Cards):** Presentación limpia de proyectos reales mediante tarjetas independientes que incluyen etiquetas de tecnologías y accesos directos.
*   **Modales Dinámicos (Modals):** Ventanas emergentes accesibles para ampliar la información técnica de cada proyecto sin saturar la vista principal.
*   **Carrusel de Imágenes (Carousel):** Deslizadores integrados dentro de los modales para mostrar capturas de pantalla de las aplicaciones de forma dinámica.
*   **Formulario de Contacto (Forms):** Sección de contacto maquetada con componentes de formulario nativos y accesibles de Bootstrap 5.
*   **Identidad Visual:** Paleta de colores personalizada mediante sobreescritura de variables nativas, adoptando un tono azul oscuro corporativo y elegante.
*   **Descarga de CV:** Botón optimizado con atributos HTML5 para la descarga directa del currículum en formato PDF.

---

## 🧰 Tecnologías Utilizadas

*   **HTML5** (Estructura semántica y accesibilidad)
*   **CSS3** (Estilos personalizados centralizados)
*   **JavaScript (ES6)** (Lógica de interactividad y control del DOM)
*   **Bootstrap v5.2.3** (Framework CSS de maquetación)
*   **Font Awesome v6.3.0** (Paquete de iconografía profesional)
*   **Google Fonts** (Tipografías *Saira Extra Condensed* y *Muli*)
